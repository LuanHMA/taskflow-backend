# TaskFlow - Auth

Aplicação fullstack moderna desenvolvida com foco em aprender autenticação e autorização utilizando JWT + OAuth em uma arquitetura profissional baseada em Next.js e Node.js.

O projeto simula um pequeno sistema de gerenciamento de tarefas, porém o foco principal está na implementação completa de autenticação moderna, integração OAuth e segurança de aplicações web.

## 🎯 Objetivo do Projeto

Este projeto foi criado com o objetivo de aprender e praticar:

- OAuth na prática
- JWT moderno
- Fluxos de autenticação reais
- Segurança em aplicações web
- Integração Frontend + Backend
- Arquitetura moderna Fullstack
- Boas práticas utilizadas no mercado em 2026

## ⚙️ Backend

API REST desenvolvida utilizando Fastify com foco em:

- JWT Authentication
- Refresh Token
- OAuth integration

- Arquitetura moderna
